### What is AI in chatbots?

AI-powered chatbots uses NLP to understand language, ML to improve over time, and integrations to fetch real data.. The key technologies that make this possible are NLP, Machine Learning, Speech Recognition, and APIs.

### How does Amazon Lex simplify chatbot development?

Building chatbots traditionally requires complex coding, NLP expertise, and integration work to understand user input and respond correctly. Amazon Lex simplifies this by providing built-in natural language understanding, speech recognition , etc..,
